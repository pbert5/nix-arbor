# `desktop/hyprland-minimal`

Minimal Hyprland desktop branch for bootstrapping and troubleshooting.

## Main Effects

- enables Hyprland through the NixOS module with XWayland and UWSM
- uses greetd with the text `tuigreet` greeter
- enables graphics, input, D-Bus, Polkit, and PipeWire audio basics
- applies a tiny Home Manager Hyprland config with Kitty, Rofi, workspace
  bindings, window movement, and media keys

This branch intentionally avoids the richer `desktop/hyprland` stack:
ReGreet, Waybar, wallpaper services, Hyprlock, Hypridle, notifications,
Bluetooth menus, printing, removable-media services, and file-manager defaults
are left out so the session is easier to debug.

Home Manager force-links `~/.config/hypr/hyprland.conf` so Hyprland's
autogenerated stub cannot block activation or become the active session config.

## Starter Bindings

- `Super+Return`: open Kitty
- `Super+D`: open Rofi
- `Super+Q`: close the active window
- `Super+Shift+E`: exit Hyprland
- `Super+V`: toggle floating
- `Super+F`: toggle fullscreen
- `Super+1` through `Super+5`: switch workspace
- `Super+Shift+1` through `Super+Shift+5`: move the active window

## Selection

Use this when the full Hyprland branch is not stable enough to get into a
session. It requires `desktop` and conflicts with the other desktop
environment sub-dendrites.
