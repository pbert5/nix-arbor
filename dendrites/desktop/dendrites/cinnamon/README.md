# `desktop/cinnamon`

Cinnamon desktop environment for workstation-class hosts.

## Purpose

Provides a full Cinnamon graphical stack on top of the `desktop` parent
dendrite.

## Main Effects

- enables X11
- enables LightDM
- enables the Cinnamon desktop session
- enables Bluetooth, graphics, PipeWire, printing, GVFS, and related desktop
  services

## Requirements

- requires `desktop`
- conflicts with `desktop/gnome`
- intended for `workstation` hosts
