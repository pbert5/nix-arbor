# `desktop/hyprland`

Hyprland desktop environment for workstation-class hosts.

## Main Effects

- enables Hyprland through the NixOS module and launches it with UWSM
- uses SDDM on Wayland and defaults to the `hyprland-uwsm` session
- enables the matching Hyprland portal plus the GTK file-picker fallback
- enables PipeWire, WirePlumber, graphics, Bluetooth, printing, removable
  storage, and the usual desktop services
- applies a shared Home Manager module to the host's users with:
  - Kitty, Rofi, Waybar, SwayNotificationCenter, Hypridle, and Hyprlock
  - Dolphin as the file manager
  - Grim, Slurp, and wl-clipboard for screenshots
  - Hyprpolkitagent, GNOME Keyring Secret Service, and Awww as
    graphical-session systemd services
  - a conservative starter Hyprland configuration and key bindings

The wallpaper daemon starts automatically, but no site wallpaper is imposed.
Set one at runtime with `awww img /path/to/wallpaper`.

Home Manager force-links `~/.config/hypr/hyprland.conf` so Hyprland's
autogenerated stub cannot block activation or become the active session config.
Keyboard shortcuts bypass application keybind inhibition and dismiss an open
Rofi menu before running their normal action.

## Important Key Bindings

- `Super` (press and release): Rofi application launcher
- `Super+Return` or `Super+K`: Kitty
- `Super+F`: Dolphin
- `Super+Shift+F`: toggle fullscreen
- `Print`: capture all monitors, save it, and copy it to the clipboard
- `Shift+Print`: capture a selected region, save it, and copy it to the clipboard
- `Super+N`: notification center
- `Super+L`: lock
- `Super+Q`: close the active window
- `Super+Up`: toggle Hyprspace's workspace overview on the current display
- `Super+Shift+Up`: toggle Hyprspace's workspace overview on all displays
- `Ctrl+Alt+Up`: toggle Hyprspace's workspace overview on all displays
- `Super+Shift+Left`/`Right` or `Ctrl+Alt+Left`/`Right`: shift the workspace
  set between displays
- `Super+1` through `Super+5`: change workspace
- `Alt+Shift+1` through `Alt+Shift+0`: move the active window to workspace
  1 through 10 and follow it
- `Super+Shift+1` through `Super+Shift+0`: move the active window silently to
  workspace 1 through 10
- `Super+Shift+Left` / `Super+Shift+Right`: shift every display's active
  workspace backward or forward by the number of connected displays

Workspace 1 defaults to the LG UltraGear monitor and workspace 2 defaults to
the Acer ED340CU monitor. Both workspaces remain present while empty.

Hypridle locks the session after ten minutes and powers off the displays one
minute later. Keyboard or pointer activity restores the displays to show
Hyprlock.

## Requirements

- requires `desktop`
- conflicts with `desktop/cinnamon` and `desktop/gnome`
- intended for workstation hosts

The current host package set supplies Hyprland and the first-party Hypr tools.
Awww comes from the repo's existing unstable overlay because the stable package
set still carries its archived predecessor, Swww.

The flake currently pins Hyprland 0.55.2. Home Manager is explicitly kept on
the native `hyprland.conf` generator because the users still have a 25.11 home
state version and the starter bindings use Hyprlang strings. Migrating this
module to Home Manager's Lua generator should be a separate, tested change.
